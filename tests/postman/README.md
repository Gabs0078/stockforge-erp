# Testes manuais sugeridos

1. Criar fornecedor com CNPJ inválido e confirmar erro.
2. Criar fornecedor com e-mail inválido e confirmar erro.
3. Criar produto com SKU X no Galpão Central.
4. Tentar criar outro produto com o mesmo SKU no mesmo galpão e confirmar bloqueio.
5. Criar produto com mesmo SKU em outro galpão e confirmar permissão.
6. Fazer entrada de estoque.
7. Fazer saída de estoque maior que saldo disponível e confirmar bloqueio.
